1. 코드의 상단 부분은 using 명령문과 변수 선언목록으로 시작합니다.

```csharp
using System;

// initialize variables - graded assignments
int currentAssignments = 5;

int sophia1 = 90;
int sophia2 = 86;
int sophia3 = 87;
int sophia4 = 98;
int sophia5 = 100;

int andrew1 = 92;
int andrew2 = 89;
int andrew3 = 81;
int andrew4 = 96;
int andrew5 = 90;

int emma1 = 90;
int emma2 = 85;
int emma3 = 87;
int emma4 = 98;
int emma5 = 68;

int logan1 = 90;
int logan2 = 95;
int logan3 = 87;
int logan4 = 88;
int logan5 = 96;
```

1. 여기에 변수 선언 코드 행의 두 코드 그룹을 추가합니다. 첫 번째 그룹은 시헙 접수의 합계를 유지하는데 사용되는 정수입니다. 두 번째 그룹은 계산된 평균 점수를 유지하는데 사용되는 소수입니다. 정수 계산은 계산된 값의 소수 부분을 반올림하기 때문에 _여기에서 코드는 소수점을 사용합니다_.

```csharp
sophiaSum = sophia1 + sophia2 + sophia3 + sophia4 + sophia5;
andrewSum = andrew1 + andrew2 + andrew3 + andrew4 + andrew5;
emmaSum = emma1 + emma2 + emma3 + emma4 + emma5;
loganSum = logan1 + logan2 + logan3 + logan4 + logan5;

sophiaScore = (decimal)sophiaSum / currentAssignments;
andrewScore = (decimal)andrewSum / currentAssignments;
emmaScore = (decimal)emmaSum / currentAssignments;
loganScore = (decimal)loganSum / currentAssignments;
```

1. 방정식의 첫 번째 그룹은 각 학생의 과제 점수 합계를 계산하는데 사용합니다. 방정식의 두 번째 그룹은 평균 점수를 계산합니다. 나누기가 분수 구성 요소를 유지하도록 하기 위해 분자가 소수로 변환됩니다.

```csharp
Console.WriteLine("Student\t\tGrade\n");
Console.WriteLine("Sophia:\t\t" + sophiaScore + "\tA");
Console.WriteLine("Andrew:\t\t" + andrewScore + "\tB");
Console.WriteLine("Emma:\t\t" + emmaScore + "\tB");
Console.WriteLine("Logan:\t\t" + loganScore + "\tA");

Console.WriteLine("Press the Enter key to continue");
Console.ReadLine();
```

1. Console.ReadLine(); 은 응용 프로그램 사용자가 출력을 검토할 수 있도록 응용프로그램을 일시 중지합니다.

## foreach 반복을 만들어 Sophia 의 성적을 계산합니다.

```csharp
int sophiaSum = 0;
int andrewSum = 0;
int emmaSum = 0;
int loganSum = 0;

decimal sophiaScore;
decimal andrewScore;
decimal emmaScore;
decimal loganScore;

sophiaSum = sophia1 + sophia2 + sophia3 + sophia4 + sophia5;
andrewSum = andrew1 + andrew2 + andrew3 + andrew4 + andrew5;
emmaSum = emma1 + emma2 + emma3 + emma4 + emma5;
loganSum = logan1 + logan2 + logan3 + logan4 + logan5;

sophiaScore = (decimal)sophiaSum / currentAssignments;
andrewScore = (decimal)andrewSum / currentAssignments;
emmaScore = (decimal)emmaSum / currentAssignments;
loganScore = (decimal)loganSum / currentAssignments;
```

1. 변수를 선언하고 합계 및 평균 점수 값에 대한 계산을 수행하는 데 사용되는 코드 라인을 찾습니다.
2. 합계 계산을 수행하는 데 사용되는 코드 줄을 삭제합니다.
3. Andrew, Emma 및 Logan에 대해 선언 `int`하고 변수를 지정하는 코드 줄을 삭제합니다 .`decimal`
4. Andrew, Emma 및 Logan의 평균 점수를 계산하는 데 사용되는 코드 줄을 삭제합니다.
5. 코드 맨 아래로 스크롤한 다음 Andrew, Emma 및 Logan의 성적을 보고하는 데 사용된 코드 줄을 삭제합니다.

```csharp
// initialize variables - graded assignments
int currentAssignments = 5;

int[] sophiaScores = new int[] { 90, 86, 87, 98, 100 };
int[] andrewScores = new int[] { 92, 89, 81, 96, 90 };
int[] emmaScores = new int[] { 90, 85, 87, 98, 68 };
int[] loganScores = new int[] { 90, 95, 87, 88, 96 };

decimal sophiaScore;

sophiaScore = (decimal)sophiaSum / currentAssignments;

Console.WriteLine("Student\t\tGrade\n");
```

1. foreachSophia 의 점수를 반복하는데 사용할 명령문을 작성하려면 다음 코드를 작성하세요

```csharp
foreach (int score in sophiaScores)
{
    // add the exam score to the sum
    sophiaSum += score;
}
```

1. 이제 코드는 다음과 유사해야합니다.

```csharp
// initialize variables - graded assignments
int currentAssignments = 5;

int[] sophiaScores = new int[] { 90, 86, 87, 98, 100 };
int[] andrewScores = new int[] { 92, 89, 81, 96, 90 };
int[] emmaScores = new int[] { 90, 85, 87, 98, 68 };
int[] loganScores = new int[] { 90, 95, 87, 88, 96 };

int sophiaSum = 0;

decimal sophiaScore;

foreach (int score in sophiaScores)
{
    // add the exam score to the sum
    //sophiaSum = sophiaSum + score;
    sophiaSum += score;
}

sophiaScore = (decimal)sophiaSum / currentAssignments;

Console.WriteLine("Student\t\tGrade\n");
Console.WriteLine("Sophia:\t\t" + sophiaScore + "\tA-");

```
