.NET 런타임은 C# 컴파일러를 사용하여 C# 코드에서 어셈블리를 만든다.

Visual Studio 2022 fmf tjfclgkdu .NET SDK 를 설치하여 키본 클래스 라이브러리라는 포괄적인 라이브러리 및 애플리케이션 프레임워크 집합을 사전설치한다.

다른 타사 라이브러리의 경우 개발자는 NuGet 같은 패키지 관리자를 이용하여 NuGet 갤러리 같은 패키지 리포지토리에서 코드 라이브러리를 찾아 통합한다.

- 어느 프레임워크가 어느 앱 모델을 지원하는지 다음 표를 통해 앱 모델을 .NET 프레임워크에 매핑한다.

## .NET 을 사용하는 방법

\_개발자가 .NET 을 사용하여 애플리케이션을 빌드하는 방법

### 개발 환경 설정 / Visual Studio 2022 설치 (best)

### 시작

- 템플릿 및 정의한 프로젝트 이름을 기반으로 폴더와 파일을 생성
- 앱 모델 프레임워크에 필요한 라이브러리와 일반적으로 사용되는 라이브러리에 대한 참조를 추가
- 애플리케이션을 실행하고 코드가 컴파일되는지를 확인할 수 있도록 작은 샘플 애플리케이션을 확인할 수 있는 필수 코드를 제공
- 코드에 지침이 포함된 경우도 있다. 이러한 지침은 샘플 애플리케이션 수정 및 자체 애플리케이션 개발을 안내한다.

### 기능 빌드

\_다음으로 , 애플리켕이션의 논리를 작성하기 시작한다. 다음과 같은 특별한 작업을 수행해야 할 때 코드 라이브러리에 참조를 추가한다.

- 네트워크 리소스에 연결
- 데이터를 저장하거나 검색하기 위해 데이터베이스에 엑세스
- 데이터를 한 형식에서 다른 형식으로 변환
- 로깅을 추가하여 애플리케이션의 문제 진단

### 애플리케이션 디버그

\_소프트웨어를 빌드하는 과정에서 애플리케이션을 실행하여 요구 사항과 기댈치를 충족하는지 여부를 측정하게 된다. 애플리케이션에서 버그를 발견하지만 발생한 이유나 이 버그를 수정할 방법을 잘 모를 수도 있다.

디버그할 때 애플리케이션이 실행되는 것을 확인하면서 상황을 파악한다.

- 애플리캐이션을 일시 중지하고 실행 시 코드를 따라갈 수 있도록 하는 중단점을 설정한다
- 변수에 저장된 값을 관찰하고, 코드의 실행 경로를 제어하여 줄을 건너뛰거나 다시 실행하고, 변수 값을 변경하는 등의 작업을 한다.
- 시각적 개체와 명령줄 디버깅 도구를 모두 사용하면 애플리케이션의 실행을 관찰하고 제어할 수 있다. 이 기능을 사용하여 애플리케이션이 실행 중인 동안 상황을 파악한다.

### 애플리케이션 배포

\_애플리케이션을 릴리스할 준비가 되면 빌드의 릴리스 버전을 만든다. 릴리스 버전에서는 디버깅에 필요한 코드를 제거한다. 또한 시각적 개체 및 명령줄 도구에서 릴리스 버전을 컴파일할 수 있다.

릴리스 버전을 실행하려면 대상 컴퓨터에 .NET 런타임을 설치해야 한다. 이 컴퓨터는 컴파일된 .NET 어셈블리를 실행한다.

### 런타임에 .NET 이 작동하는 방법

애플리케이션의 .NET 어셈블리 및 .NET 런타임을 대상 컴퓨터에 설치한 후에서는 애플리케이션을 실행할 수 있다.

.NET 런타임은 애플리케이션에 대한 실행 환경을 제공하는 보호막과 같다.

- 프로그램을처음 실행할 때 중간 코드를 이진 형식으로 컴파일한다. 이진 형식은 실행 중인 컴퓨터의 플램폼 및 아키텍처 (예: 64비트 Windows) 와 관련이 있다.
- 프로그램의 진입점을 찾아서 각 명령을 적절한 시퀀스로 실행하기 시작한다
- 메모리, 네트워크 액세스와 같은 컴퓨터 리소스를 관리한다. .NET 런타임이 “메모리를 관리한다”는 말은 운영 체제를 사용하여 애플리케이션에 맞게 메모리를 \*프로비저닝 한다는 의미다 \*프로비저닝(provisioning)은 **사용자의 요구에 맞게 시스템 자원을 할당, 배치, 배포해 두었다가 필요 시 시스템을 즉시 사용할 수 있는 상태로 미리 준비해 두는 것을 말한다**
- 애플리케이션에 더 이상 메모리에 저장된 데이터가 필요하지 않은 경우 가비지수집 기능은 소프트웨어 개발자의 지침 없이 해당 메모리를 운영체제로 다시 릴리스한다.
- 잠재적으로 악의적인 의도를 가진 소프트웨어로부터 사용자의 컴퓨터를 보호한다. 또한 애플리케이션 간의 격리 레이어를 제공한다.
