### avaScript 에서 콜백의 의미

<aside>
💡 JavaScript 의 이벤트는 모두 싱글 스레드압나더 * 한 번에 한 가지 일만 처리합니다. 그래서 콜백함수가 등장하게 되었습니다. 작업을 요청한 이후에 그 작업이 끝날 때까지 다음 작업은 진행할 수 없습니다. 
⇒ 굉장히 큰 자원낭비가 발생하기 때문에, 이를 해결하기 위해 콜백함수가 대안이 되었다. ‘비동기’적인 작업이 가능하게 된것입니다, 
어떤 함수를 실행시키고 난 이후에 결과를 받을 함수, 혹은 그 다음에 실행될 함수를 콜백함수라고 합니다. 
*setTimeOut() 과 같은 함수에서 어떻게 사용되는지 알아보도록 합시다.

</aside>

### 먼저 콜백을 사용하지 않은 코드를 만들어보겠습니다.

```jsx
const 폴짝 = (count) => {
  console.log(`땃쥐가 ${count}회 폴짝 뛰어올랐습니다`);
};
```

### 횟수를 입력받으면 횟수만큼 땃쥐가 폴짝 하고 뛰어오르는 폴짝 함수입니다.

이제 콜백함수를 만들어서 폴짝 함수를 전달해보도록 하겠습니다.

---

### 흔들흔들함수 → 폴짝 함수

```jsx
const 폴짝 = (count) => {
 console.log(`땃쥐가 ${count}회 폴짝 뛰어올랐습니다`)
}

const 흔들흔들 = (count, 폴짝) => {
 console.log(`안녕!! ${count}번 땃쥐가 코를 흔들었습니다.`
 폴짝();
}
```

<aside>
💡 땃쥐는 count번 입력받은대로 안녕! 하고 코를 흔든 뒤 count 번 폴짝 뛰게 됩니다.

</aside>

---

### setTimeout()

```jsx
setTimeout(흔들흔들, 3000, 폴짝);
```

<aside>
💡 땃쥐는 원하는 시간 이후에 폴짝을 실행할 수 있습니다.
흔들흔들을 실행한 이후 3초간 휴식 후에 폴짝을 실행하게 됩니다.

</aside>
