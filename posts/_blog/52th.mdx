### Alias type

### `type Player`

```tsx
type Age = number;
type Name = string;
type Player = {
  	name:string,
    age?:number
}
//age 뒤에 ? 을 씀으로서 optional이 되었다.
const jaHyeon : Player={
	{
	name:"jahyeon"
}

const nico Player={
	{
	name:"nico"
  age:12
}
```

### never

> 함수가 절대 return 하지 않을때 발생한다.

```tsx
function hello(name: string | number) {
  if (typeof name === 'string') {
    name;
  } else if (typeof name === 'number') {
    name;
  } else {
    name;
  }
}
```
