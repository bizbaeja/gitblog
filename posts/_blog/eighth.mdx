### 🥨 XHTML 역할의 모듈 항목

- banner - 페이지의 주요 제목 또는 내부 제목을 포함합니다.
- complementary - “보완” 이라는 뜻으로, DOM 계층 구조에서 유사한 수준의 기본 콘텐츠를 보완하도록 설계된 문서의 지원 섹션이지만 기본 콘텐츠와 분리될 때의 의미를 유지한다고 합니다.
- contentinfo - contentifo 에는 페이지 또는 전체 페이지의 콘텐츠에 대한 **메타 정보**가 있습니다.

\*메타 정보\_ : 데이터에 관한 구조화된 데이터로, ,다른 데이터를 설명해 주는 데이터

- definition : 정의는 용어 또는 개념의 정의를 나타냅니다.
- main : main 은 문서의 주요 내용으로 작동합니다.
- navigatioin : 탐색은 문서 또는 관련 문서 탐색에 적합한 항목 모음을 나타냅니다
- note : note 는 콘텐츠가 리소스의 주요 콘텐츠에 대해 괄호 안에 있거나 보조적임을 나타냅니다.
- search : search 는 섹션이 검색 기능을 제공함을 나타냅니다.
- alert : 경고 또는 오류 정보가 포함된 메시지 입니다.
- alertdialog : 경고 또는 오류가 있는 별도의 창입니다.
- application : 사용자를 위해 일련의 작업을 실행하는 소프트웨어 단위입니다.
- button : 사용자 트리거 작업을 허용합니다.
- checkbox : 세 가지 가능한 값 ( true, false, 혼합) 이 있는 컨트롤입니다.
- columnheader : 열에 대한 헤더 정보가 포함된 테이블 셀입니다.
- combobox : 콤보 상자는 사용자가 입력하여 선택한 항목을 찾을 수 있는 선택의 프레젠테이션입니다.
- description : describeby 를 통해 이 요소를 참조하는 페이지 요소에 대한 설명 콘텐츠입니다.
- dialog : 대화 상자는 응용 프로그램 위에 있는 작응 응용 프로그램 창으로, 사용자에게 정보를 입력하거나 응답을 용구하도록 프롬프트하기 위해 응용 프로그램의 현재 처리를 중단하도록 설계되었습니다.
- directory : 단일 그룹의 구성원에 대한 참조 목록입니다.
- document : 책과 같은 관련 정보가 포함된 콘텐츠입니다.
- grid :그리드는 행과 열로 정렬된 테이블 형식의 데이터의 셀을 포합합니다.
- girdcell: 그리드셀은 그리드의 테이블 셀입니다 그리드 셀은 활성 상태이고 편집 가능하며 선택 가능합니다. 셀에는 기능적 관계ㅈ의 적용을 처리하기 위한 컨트롤과 같은 관계가 있을 수 있습니다.
- group : 그룹은 보조 기술에 의해 페이지 요약이나 목차에 포함되니 않는 사용자 인터페이스 개체의 섹션입니다. 페이지 요약 또는 목차에 포함되어야 하는 사용자 인터페이스 개체 섹션은 지역을 참조하십시오.
- heading : 페이지 섹션의 제목입니다.
- img : img는 이미지를 구성하는 컬렉션 요소의 컨테이너입니다.
- link : 리소스에 의한 대화식 참조
- list : 비대화형 목록 항목 그룹입니다. 목록에는 역할이 listitem 인 자식이 포함됩니다.
- listbox : 목록 상자는 사용자가 목록에서 하나 이상의 항목을 선택할 수 있도록 하는 위젯입니다. 목록 내의 항목은 정적이며 이미지를 포함할 수 있습니다. 목록 상자에는 역할이 옵션인 자식이 포함됩니다.
- listitem : 목록의 단일 항목입니다.
- log : 채팅, 메시지, 게임로그, 에러 로그 등 새로운 정보가 추가되고 기존 정보가 사라질 수 있는 영역입니다. 다른 지역과 달리 이 역할에서는 로그의 새 항목 도착과 읽기 순서 사이에 관계가 있습니다 로그에는 의미 있는 시퀀스가 포함되어 있으며, 임의의 지점이 아닌 로그 끝에만 새 정보가 추가됩니다.
- marquee : 선택 윤곽은 페이지에서 텍스트를 스크롤하는 데 사용됩니다.
- menu : 사용자에게 선택 목록을 제공합니다.
- menubar : 메뉴바는 메뉴 항목의 컨테이너입니다. 각 메뉴 항목은 새 하위 메뉴늘 활성화할 수 있습니다. 탐색 동작은 일반적인 메뉴 모은 그래픽 사용자 인터페이스와 유사해야 합니다.
- menuitem : 메뉴의 링크입니다. 이것은 메뉴에 포함된 선택 그룹의 옵션입니다.
- menuitemcheckbox: 체크 가능한 메뉴 항목을 정의합니다.
- option : 선택으로 표시되는 목록에서 선택 가능한 항목입니다.
- presentation : 역할이 프리젠테이션인 요소는 접근성 API 에 매핑할 필요가 없습니다.
- progressbar : 실행 진행률을 표시하기 위해 실행 시간이 오래 걸리는 작업에 응용 프로그램에서 사용합니다.
- radio : 라디오는 단일 선택 목록의 옵션입니다. 라디오 그룹에서 동시에 하나의 라디오 컨트롤만 선택할 수 있습니다.
- regeion : 지역은 웹 페이지에서 감지할 수 있는 큰 섹션입니다.
- row : 테이블 셀의 행입니다.
- rowheader: 행에 대한 헤더 정보가 포함된 테이블 셀입니다.
- separator : 콘텐츠의 섹션을 구분하고 구분하는 선 또는 막대입니다.
- slider : 사용자가 주어진 범위에서 입력을 선택하는 사용자 입력입니다. 이러한 형태의 범위는 아날로그 키보드 인터페이스를 기대합니다.
- status : 이것은 사용자에게 피드백을 제공하기 위한 프로세스 권고 정보의 컨테이너입니다.
- tab : 탭 패널의 헤더입니다.
