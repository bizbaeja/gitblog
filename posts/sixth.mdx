### 동일한 종류의 객체를 여러 개 생성해야 하는 경우 (코딩애플 예시 참고)

```
//object 형식으로 ...!
var캐릭터1={여기에 챔마다 정보를 담을건데}
var nunu = {
 q : 'consume'
 w : 'snowball'
}
var garen = {
 q : 'strike'
 w : 'courage'
}
....
이렇게 비슷한 object를 많이 만들 일이 잇으면 class를 만들어 쓰는 것이다.

```

⇒ new 연산자와 생성자 함수에서 배운 new function 을 사용할 수 있다.

> 여기에 어해 모던 자바스크립트에 도입된 클래스(class) 라는 문법을 사용하면 객체 지향 프로그래밍에서 사용되는 다양한 기능을 자바스크립트에서도 사용할 수 있습니다.

### 기본 문법

⇒ 클래스는 다음과 같은 기본 문법을 사용해 만들 수 있습니다.
this 란? 이 클래스로부터 생성되는 object 입니다. (멋진말로 instance)

```
class Hero {
    constructor(구멍){
        this.q = 구멍;
        this.w = 'snowball';
    }
}
// constructor(){} 안에 this.어쩌고 넣는다.
```

이렇게 클래스를 만들고 Hero 를 호출하면 내부에서 정의한 메서드가 들어있는 객체가 생성됩니다.
객체의 기본 상태를 설정해주는 생성자 메서드 constructor() 는 new 에 의해 자동으로 호출되므로 , 특별한 절차 없이 객체를 초기화 할 수 있습니다.
(예시)

```
class Hero {
    constructor(name){
        this.name = name;
        }
        sayHi(){
            alert(this.name);
        }
}
//사용법:
let hero = new Hero("Nunu");
hero.sayHi();
```
