### 왜 Node JS를 사용하는가?

<aside>
✍🏻   Node JS는 브라우저 밖에서 돌아가는 자바스크립트를 사용하는 이점을 사용하고 싶어서 만들어진 프로그램입니다. 자바스크립트는 브라우저없이는 돌아가지 않거든요. 이제는 Node JS 덕분에 브라우저 없이도 자바스크립트를 프로그래밍 할 수 있게 되었습니다.

</aside>

---

### Node Js 설치하기

[다운로드 | Node.js](https://nodejs.org/ko/download/)

---

### 1. 터미널을 실행합니다

### 2. brew를 실행합니다

```jsx
$ brew install node
```

### 3. 노드 버전을 확인하여. path 지정할 버전을 기억해두어야 합니다.

```jsx
$ node -v
```

### 4. bash 파일이 있으면 수정하고 없으면 추가하여 path를 설정해줍니다.

```jsx
cd ~
vi .bash_profile
```

### 5. 간단히 알아보는 vi 컨트롤법

- 아래 코드를 입력합니다.

<aside>
✍🏻 파일의 끝으로 이동할 때 : G
한글자 삭제 : x
단어 삭제 : dw
실행 취소 : u
줄의 맨 앞 : o
줄의 맨 뒤 : $
현재 커서에 삽입 : i
현재 커서 바로 다음에 삽입 : a 
——————— 
(esc 종료 모드 )
저장만 : w 
종료만 : q
저장 후 종료 :wq

</aside>
