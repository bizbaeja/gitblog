## 프록시란?

- 프록시란 사용자의 요청이 갈 수 있는 최적의 경로를 찾아, 해당 서버에 보내고 그 결과를 사용자에게 대신 전달해주는 것.

## HTTP 와 HTTPS 의 차이

- HTTPS 프로토콜은 SSL (보안 소켓 계층) 을 사용함으로서 서버와 브라우저 사이에 안전하게 암호화된 연결을 만들 수 있게 도와주고, 서버 브라우저가 민감한 정보를 주고받을 떄 이것이 도난당하지 않게 암호화해준다. SEO 최적화에 유리하다, 사용자들이 결국에는 가장 안전하다고 생각하는 사이트를 더 많이 방문하기 때문에

## SPA 란

- 단일 페이지 어플리케이션은 현재 웹 개발의 트렌드 브라우저에 최초에 한번 페이지 전체를 로드하고, 이후부터는 특정 부분만 Ajax를 통해 데이터를 바인딩하는 형식이다.

## 자바스크립트와 관련한 same-origin 정책

- 동일 출처 원칙 은 자바스크립트가 서로 다른 도메인 사이에 리퀘스트를 날리는 것을 방지한다.랜섬웨어같은 악성 프로그램의 삽입을 방지하며, 웹페이지에서 DOM 을 통해 민감한 정보를 가져가는 것을 방지한다.

## Webpack 이란?

- html 파일에 들어가는 자바스크립트 파일들을 하나의 자바스크립트 파일로 만들어주는 것

## 스코프란?

- 변수에 접근할 수 있는 범위
  전역 변수 : 어디에서든 사용할 수 있는 변수
  지역 변수 : 함수 스코프에서만 사용할 수 있는 변수

## 이벤트 위임이란?

- 부모 요소에 이벤트 리스너를 붙이는 것
  하위 요소에 이벤트가 발생하면 이벤트 버블링 때문에 부모 요소에연결된 리스너가 실행됨
  (장점) 사용하는 메모리의 양이 감소한다. 모든 요소마다 리스너를 추가하는게 아니라 부모에만 추가하면 되기 때문에.

## this 용법에 대해서 설명하시오

- this 는 자신이 속한 객체 또는 자신이 생서알 인스턴스를 가리키는 자기 참조 변수
  this 는 함수의 호출 방씩에 따라 특정 객체를 바인딩

## 시멘틱 태그란?

- 의미가 있는 태그라는 뜻 , HTML 태그를 통해 웹 문서를 작성할 때 좀 더 의미가 있는 태그를 사용함으로서 정확하게 정보를 전달하고 웹 문서의 구조를 쉽게 파악할 수 있도록 합니다.

## 면접에서 모르는 질문 나왔을 때

- 질문을 생각해 볼 시간을 달라고 한다. 제가 생각해보지 못한 질문이네요 제가 정리할 시간이 필요할 것 같은데 1~2분 정도 생각해 볼 수 있을까요?

## 온프레미스

- 소프트웨어 등 솔루션을 클라우드같이 원격환경이 아닌 전산실 서버에 직접 설치해 운영하는 방식 (클라우드와 반대되는 개념)

## 클라우드

- SaaS : 소프트웨어 (예시: 구글)
  PaaS : 플랫폼 (예시 : 마이크로소프트 애저)
  IaaS : 인프라스트럭쳐 (예시 : 아마존 AWS 응용서버 웹서버 운영을 위한 가상환경 제공)

## EC2

- AWS 에서 사용자가 인스턴스라고 불리는 가상 컴퓨터를 임대받는 서비스

## S3

- 무한대의 객체를 저장할 수 있는 AWS 의 서비스
